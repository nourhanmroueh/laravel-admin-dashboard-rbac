# Laravel Admin Panel

A clean and extensible Laravel 12 admin panel with role-based access control,
products & categories management, and AdminLTE UI.

## Features
- Authentication
- Roles & permissions (Spatie)
- Products & categories CRUD
- Soft delete & restore
- Filters & search
- AdminLTE UI

## Tech Stack
- Laravel 12
- PHP 8.3
- MySQL
- AdminLTE
- Spatie Laravel Permission

## Installation
```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate:fresh --seed
php artisan serve
